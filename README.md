Statistics

### Area for normal distribution of s.d.

  - +/- s.d. 68%
  - +/- 2*s.d. 95%
  - +/- 3*s.d. 99.7%
  
### ROC

ROC curve, is a graphical plot that illustrates the diagnostic ability of a binary classifier system as its discrimination threshold is varied.

  -  Y - TPR, X - FPR
  - TPR = TP / (TP + FN) = TP / P
  - FPR = FP / (FP + TN) = FP / N
